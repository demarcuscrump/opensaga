# OpenSaga — Copywriting & Voice Guide

**Every word in the platform is a design decision.**  
This guide ensures OpenSaga speaks with one voice — literary, empowering, precise, and slightly mythic.

---

## Brand Voice

### The Four Pillars

| Pillar | Meaning | Do | Don't |
|---|---|---|---|
| **Literary** | We speak like storytellers | "Forge a universe from nothing" | "Create a new project" |
| **Empowering** | The creator is the hero | "Your world. Your rules. Your canon." | "We help you collaborate better" |
| **Precise** | No fluff, no jargon | "The community votes. If 60% approve, it becomes Canon." | "Leverage our decentralized governance synergy" |
| **Mythic** | A hint of epic without cringe | "The Lorekeeper Council has spoken." | "Your submission was approved by the moderators!" |

### The Line We Never Cross

Mythic ≠ cringe. We are **slightly** epic. Think of a narrator with a quiet voice, not a stadium announcer. The moment something reads like a mobile game push notification ("LEGENDARY LOOT AWAITS!!!"), it's wrong.

## Platform Vocabulary

Every feature in OpenSaga has an in-world name. Use these consistently — never the generic equivalents.

| OpenSaga Term | Generic Equivalent | Usage |
|---|---|---|
| **World** | Project, Space, Community | "Explore Worlds" not "Browse projects" |
| **World Bible** | Rulebook, Guidelines, Wiki | "Consult the World Bible" not "Check the rules" |
| **Canon** | Approved, Published, Official | "Canonized" not "Approved" |
| **Proposal** | Submission, Entry, Draft | "Submit a Proposal" not "Post your idea" |
| **Forge** | Create, Build, Make | "Forge a character" not "Create a character" |
| **Canonize** | Approve, Accept | "The community canonized your creation" |
| **Lorekeeper** | Moderator, Admin, Curator | "Lorekeepers review proposals" |
| **Architect** | Power user, Top contributor | "Promoted to Architect" |
| **Citizen** | Member, User, Participant | "42 Citizens" not "42 members" |
| **The Saga** | Activity feed, Timeline | "See what's happening in The Saga" |
| **Creator Studio** | Editor, Workspace, Tool | "Open the Creator Studio" |
| **Character Forge** | Character creator | "Enter the Character Forge" |
| **Canon Check** | Validation, Review | "Run a Canon Check" not "Validate your submission" |
| **World Seed** | Template, Starter | "Plant a World Seed" |
| **The Oracle** | AI assistant | "Consult the Oracle" (for AI features) |

## Microcopy Patterns

### Buttons

Always a verb. Short. Commanding.

| Context | Good | Bad |
|---|---|---|
| Primary CTA | "Forge World" | "New World" |
| Secondary CTA | "Explore Worlds" | "See All" |
| Submit action | "Submit Proposal" | "Done" / "Submit" |
| Destructive | "Abandon Draft" | "Delete" |
| AI trigger | "Consult the Oracle" | "Generate" |
| Save | "Save to Drafts" | "Save" |
| Navigation | "Enter World" | "Open" / "View" |
| Publish | "Publish to Canon" | "Post" |
| Vote | "Cast Vote" | "Vote" |

### Empty States

Tell a micro-story. Always include a CTA.

```
Characters tab (empty):
"No characters inhabit this world yet. Be the first to breathe life into it."
[Forge a Character]

Worlds (empty explore):
"The multiverse is quiet. No worlds have been forged yet."
[Forge the First World]

Drafts (empty):
"Your workshop is empty. Every saga starts with a single spark."
[Open Creator Studio]

Activity (empty):
"Nothing has stirred in The Saga. Forge something worth talking about."
[Explore Worlds]

Search (no results):
"The archives hold no records matching your query. Try different terms, or forge what you're looking for."
```

### Loading States

Contextual, never generic.

| Context | Loading Text |
|---|---|
| AI generating | "Consulting the Oracle..." |
| Loading worlds | "Mapping the multiverse..." |
| Loading characters | "Summoning the roster..." |
| Saving draft | "Preserving your work..." |
| Submitting proposal | "Presenting to the council..." |
| Tallying votes | "The community deliberates..." |
| Loading profile | "Retrieving your legend..." |

### Success Messages

Brief. Celebratory but dignified.

| Context | Message |
|---|---|
| World created | "A new world emerges. Shape its destiny." |
| Character forged | "Your creation draws its first breath." |
| Proposal submitted | "Your proposal is before the community. Voting begins now." |
| Canon achieved | "Canonized. This creation lives forever in the world's history." |
| Vote cast | "Your voice has been heard." |
| Draft saved | "Draft preserved." |
| Profile updated | "Legend updated." |

### Error Messages

Apologetic. Never blame the user. Always offer a next step.

| Context | Message |
|---|---|
| AI failure | "The Oracle is resting. Try again in a moment." |
| Network error | "Lost connection to the archives. Check your network and retry." |
| Auth required | "This path requires a known identity. Sign in to continue." |
| Permission denied | "You don't have standing in this world. Request membership first." |
| Vote already cast | "You've already spoken on this proposal." |
| Invalid input | "The [field] needs more substance. [specific requirement]." |
| World Bible conflict | "This conflicts with the World Bible. Consult the [section] and revise." |

## Tone Calibration

### Too Corporate (Wrong)
> "Welcome to OpenSaga, a collaborative platform enabling teams to co-create fictional universes with integrated governance workflows and AI-assisted content generation."

### Too Casual (Wrong)
> "Hey! Ready to build some awesome worlds with your friends? Let's gooo!"

### Too Epic (Wrong)
> "BEHOLD, MORTAL! THE INFINITE TAPESTRY OF CREATION AWAITS YOUR DIVINE HAND! FORGE WORLDS THAT SHAKE THE VERY FOUNDATIONS OF REALITY!"

### Just Right (OpenSaga Voice)
> "Forge Universes. Govern Canon. OpenSaga is where communities collaboratively build, vote on, and evolve shared fictional universes."

## Writing Checklist

Before shipping any copy, ask:

- [ ] Would a storyteller say this? (Not a product manager)
- [ ] Does it use OpenSaga vocabulary? (Not generic SaaS terms)
- [ ] Is the user the hero? (Not the platform)
- [ ] Is it under 20 words? (For UI copy — shorter is always better)
- [ ] Would it sound good read aloud? (No awkward phrasing)
- [ ] Is it free of emojis? (We don't use emojis in the product UI)
